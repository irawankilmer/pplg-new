## ğŸª 2. **Basis Data**

*â€œGudang penyimpanan barang dan transaksiâ€*

Coba bayangkan sebuah toko, tapi gak punya tempat nyimpan barang. Atau semua stok dan catatan transaksi cuma ditulis di kertas.

* Desain struktur database yang efisien

Di marketplace nanti, kalian akan pakai database untuk menyimpan data produk, akun pengguna, transaksi, dan lainnya.
